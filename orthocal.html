Calendar project

Orthocal

An Orthodox calendar project for feasts, fasts, remembrances, with Linux/BSD CLI & web frontend, and JSON API access.

Overview

Orthocal provides a foundation for Orthodox calendar access. It is suited to static pages, web frontends, command line use, with JSON/API integrations where structured calendar data is needed.

The project can support fixed feasts, movable feasts, fasts, and days of remembrance. Public Orthodox Zone pages can later use Orthocal as the source for calendar views while keeping the hub static and fast where possible.

API and web use

Orthocal can support JSON/API use and web frontend use. That makes it suitable for simple static pages, server-rendered views, or other tools that need predictable calendar data.
